# WFC3 PSF Modeling Toolkit

A Python package for modeling and analyzing Point Spread Functions (PSFs) for the Hubble Space Telescope's Wide Field Camera 3 (WFC3).

## Features

- PSF model generation for WFC3 UVIS and IR channels
- PSF fitting and characterization  
- Comparison with empirical PSFs
- Visualization tools for PSF analysis
